# log-canvas Requirements

## 概要
- ログ解析を行う際にログビュワー上でログに対してメモやマーキングを行うことで調査効率の向上を図る。
- スクリーンショット＋外部メモツールの行き来をなくし、一つのアプリで「読む・考える・記録する」を完結させる。

## ターゲットユーザー
- ログを解析するエンジニア(ファーストステップとしてDLTログを想定)
- 調査中にメモを残したいユーザー

## コンセプト・概要
- UI/UXはモダンにする
- 無限ホワイトボードの空間にログファイルを取り込むことでホワイトボード内のカードとしてログviewが表示される
- ログview
  - 通常のログビューワ同様の機能は備えており解析が可能（DLT Viewerを想定）
  - viewのサイズを自由に変更可能であり、もちろんログは長いのでスクロール可能
  - カラムなどに対して簡単にフィルタリング/マーキング可能
- キャンバス内に自由に付箋のようなコメントを貼り付けることができる
- ログの各行の横Xピクセルに対してコメントを置くと行に紐づいてコメントの位置が固定＋ログ行の高さがコメントの高さに合わせて自動で変化する


## 機能要求

### 1. キャンバス

| ID | 要求 |
|----|------|
| C-01 | 無限スクロール可能な2Dホワイトボードを表示できる |
| C-02 | マウスホイールまたはピンチ操作でズームイン/アウトできる |
| C-03 | ドラッグ操作でキャンバスをパン（視点移動）できる |
| C-04 | キャンバス上のオブジェクト（ログビュー・コメント）を自由に移動できる |
| C-05 | 複数のログビューとコメントを同一キャンバス上に共存させられる |

### 2. ログビュー

| ID    | 要求                                                                               |
| ----- | ---------------------------------------------------------------------------------- |
| LV-01 | DLTファイルを読み込み、キャンバス上にカード（ログビュー）として表示できる          |
| LV-02 | ログビューはタイムスタンプ・ECU ID・App ID・Context ID・メッセージ等のカラムを持つ |
| LV-03 | ログビュー内を縦方向にスクロールできる                                             |
| LV-04 | ログビューのサイズ（幅・高さ）をリサイズハンドルで自由に変更できる                 |
| LV-05 | カラム単位でフィルタリング（値の一致・正規表現等）を適用できる                     |
| LV-06 | 任意のログ行をマーキング（色付きハイライト）できる                                 |
| LV-07 | カラムの表示/非表示・順序を変更できる                                              |
| LV-08 | その他一般的なViewer(DLT Viewer)の持つ機能を保有する                               |

### 3. コメント（付箋）

| ID | 要求 |
|----|------|
| CM-01 | キャンバス上の任意の位置に付箋コメントを作成できる |
| CM-02 | 付箋コメントのテキストを編集・削除できる |
| CM-03 | ログ行の右端からX px以内にコメントを配置すると、そのログ行に自動的に紐付けられる |
| CM-04 | ログ行に紐付けられたコメントは、行のスクロールに追従して位置が固定される |
| CM-05 | ログ行に紐付けられたコメントの高さに合わせて、対応するログ行の高さが自動で変化する |
| CM-06 | 紐付け済みコメントを行から切り離し、フリーなコメントに戻すことができる |

### 4. プロジェクト管理

| ID | 要求 |
|----|------|
| PM-01 | キャンバスの状態（ログビューの位置・サイズ・フィルタ、コメントの内容・位置・紐付け情報）をファイルに保存できる |
| PM-02 | 保存したプロジェクトファイルを読み込み、作業状態を復元できる |

## 技術スタック
- **フレームワーク**：Tauri
- **フロントエンド**：React + TypeScript (Vite) + Tailwind CSS
- **キャンバスライブラリ**：React Flow
- **バックエンド**：Rust
